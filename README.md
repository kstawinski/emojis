## Features

## Commands
### Project setup
If you want to run project, just clone this repository and run `serve` command.
```
git clone https://github.com/kstawinski/emojis.git
cd emojis
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
